HOW TO RUN
================================================================================

DEVELOPMENT MODE:
─────────────────
1. npm install
2. cp .env.example .env.local
3. npm run dev
4. Open http://localhost:3000
5. Login with: admin@lexaxiom.com / admin123

PRODUCTION BUILD:
─────────────────
1. npm run build
2. npm start
3. Server runs on http://localhost:3000

DOCKER DEPLOYMENT:
──────────────────
1. docker-compose up
2. Access app at http://localhost:3000
3. PostgreSQL at localhost:5432
4. Redis at localhost:6379

VERCEL DEPLOYMENT:
──────────────────
1. Push code to GitHub
2. Connect to Vercel
3. Add environment variables
4. Deploy automatically
